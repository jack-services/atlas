# Research/Discovery Issue Template

Use this template when Atlas creates research or discovery issues.

## Template

```markdown
## Context

{What problem or opportunity prompted this research}

## Research Questions

1. {Question 1}
2. {Question 2}
3. {Question 3}

## Scope

### In Scope

- {What this research should cover}

### Out of Scope

- {What this research should NOT cover}

## Tasks

- [ ] Research {topic 1}
- [ ] Research {topic 2}
- [ ] Evaluate options/approaches
- [ ] Document findings
- [ ] Present recommendations

## Deliverables

- [ ] Summary document with findings
- [ ] Recommendation with rationale
- [ ] Follow-up issues if needed

## Acceptance Criteria

- [ ] All research questions answered
- [ ] Options evaluated with pros/cons
- [ ] Clear recommendation provided
- [ ] Findings documented in knowledge repo

## Resources

### Relevant Documentation

- {Link or path to relevant docs}

### Similar Prior Work

- {References to past decisions or research}

### External Resources

- {Relevant external documentation or tools}

## Technical Notes

### Relevant Code Areas

- `{path/to/relevant/code}` - {why relevant}

### Constraints

- {Any known constraints to consider}

## Labels

- `type:research`
- `{area label}`

---
*Created by Atlas /plan command*
```

## Usage Guidelines

1. **Scope**: Be clear about boundaries to prevent scope creep
2. **Questions**: Frame specific, answerable questions
3. **Deliverables**: Define concrete outputs expected
4. **Time-box**: Consider adding a time estimate to keep focused
